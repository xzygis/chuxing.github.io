如何通过GitHub+Hexo搭建博客

Anonymous 收录于 博客
2020-01-27 2022-12-10 约 300 字 预计阅读 1 分钟 - 次阅读   - 条评论  
目录
  • 安装Node
  • 安装Hexo
  • 初始化Hexo
  • 生成网页文件和开启服务器
  • 关联Github
  • 配置next主题

安装Node 

1

$ brew install node

安装Hexo 

1

$ npm install -g hexo

初始化Hexo 

1

$ hexo init

生成网页文件和开启服务器 

1
2

$ hexo g
$ hexo s

关联Github

  1. 修改_config.yml，修改deploy为：
1
2
3
4

deploy:
  type: 'git'
  repository: https://github.com/xzygis/xzygis.github.io.git
  branch: master
  1. 生成静态文件并上传Github
1
2

$ hexo g
$ hexo d

若执行hexo d出错则执行npm install hexo-deployer-git --save。

执行hexo d会提示输入用户名密码。

配置next主题

在blog目录下执行

1

git clone --branch v5.1.4 https://github.com/iissnan/hexo-theme-next themes/next

修改_config.yml，设置theme为next。
